# Fikri Zufri — Minimalist Modern Portfolio Website

Website portofolio pribadi dengan konsep **Minimalist Modern**.

## Struktur File

- `index.html`
- `style.css`

## Cara Menjalankan

Buka file `index.html` langsung di browser.

## Edit Data

Ubah informasi profil, pengalaman, project, dan kontak langsung di file `index.html`.

## Deployment

Bisa diupload ke:
- GitHub Pages
- Netlify
- Vercel
- Hosting cPanel
- VPS Nginx
